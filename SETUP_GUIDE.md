
# 🎨 Character Copying Application 導入ガイド (初心者向け)

このガイドでは、プログラミングが初めての方でも、自分のPCでこの「模写練習アプリ」を動かすための手順を世界一わかりやすく説明します。

---

## 🛠️ ステップ1：必要なソフトを入れる（所要時間：5分）

まずは、アプリを動かすための「土台」となる2つの無料ソフトをインストールします。

1.  **Node.js (ノード・ジェーエス)**
    *   アプリを動かすためのエンジンです。
    *   [公式サイト](https://nodejs.org/ja)から「LTS (推奨版)」をダウンロードして、すべて「次へ」でインストールしてください。
2.  **Visual Studio Code (VS Code)**
    *   コードを読み書きするための高機能なメモ帳です。
    *   [公式サイト](https://code.visualstudio.com/)からダウンロードしてインストールしてください。

---

## 📂 ステップ2：プロジェクトの準備（所要時間：3分）

1.  デスクトップなど好きな場所に、新しいフォルダを作ります（名前は `art-tutor` など）。
2.  VS Codeを開き、左上の「ファイル」→「フォルダーを開く」で、今作ったフォルダを選択します。
3.  このアプリのソースコード（`App.tsx`, `index.html` など）を、すべてそのフォルダの中に保存します。

---

## ⌨️ ステップ3：魔法のコマンドを打つ（所要時間：3分）

VS Codeの画面上部にある「ターミナル」→「新しいターミナル」をクリックして、画面下に出てきた黒い画面に以下の文字をコピー＆ペーストしてEnterキーを押してください。

```bash
# アプリを動かすのに必要なパーツを自動でダウンロードします
npm install
```

---

## 🔑 ステップ4：APIキーを設定する（所要時間：2分）

AIに絵を添削してもらうための「パスポート」を設定します。

1.  VS Codeで、新しく `.env` という名前のファイルを作ります。
2.  その中に以下の1行を書き込みます（`YOUR_KEY_HERE` の部分を、自分で取得したキーに書き換えてください）。

```text
API_KEY=YOUR_KEY_HERE
```

※ APIキーは [Google AI Studio](https://aistudio.google.com/app/apikey) で無料で取得できます。

---

## 🚀 ステップ5：アプリを起動する！（所要時間：2分）

再びターミナル（黒い画面）に以下のコマンドを打ちます。

```bash
npm run dev
```

しばらくすると `http://localhost:5173` のような文字が表示されるので、それをCtrlキーを押しながらクリックするか、ブラウザのアドレス欄に直接打ち込んでください。

**おめでとうございます！これであなたの専用お絵描き教室がオープンしました！**

---

### 💡 うまくいかないときは？
*   **「コマンドが見つかりません」と出る場合**: Node.jsのインストール後に一度VS Codeを再起動してみてください。
*   **AIが返事をしない場合**: `.env` ファイルの名前が間違っていないか、キーが正しいか確認してください。
